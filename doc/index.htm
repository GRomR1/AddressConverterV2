Загрузка адресных таблиц с заявками

1. Для загрузки Excel-файла содержащего адреса для которых необходимо найти соответствие с адресами в базе данных нажмите кнопку «Открыть».

Примечание:
1) Первой строка таблицы должна содержать названия столбцов.
2) Названия столбцов должны соответствовать принятым обозначениям:
- FSUBJ - название области, края, республики и пр.
- DISTRICT - название района
- CITY1 - название города, поселка, деревни и пр.
- CITY2 - название дополнительного города, поселка, деревни и пр.
- STREET - название улицы, проспекта и пр.
- BUILD - номер дома
- KORP - номер корпуса
- LITERA - значение литеры
3) Обязательным является столбец STREET, который может содержать все остальные элементы адреса.
4) Не рекомендуется загружать таблицы содержащие более 1000 строк, лучше разбить их на несколько таблиц.

2. Укажите файл и дождитесь окончания загрузки.


3. Проверьте правильность автоматического определения составляющих адреса из загруженной таблицы. При необходимости скорректируйте значения данных полей в форме, расположенной в правой части экрана, либо в самой ячейке таблицы.

4. Если база данных загружена, то произведите поиск в ней, нажав на кнопку «Поиск».

5. В процессе поиска адресам, которые имеются в базе будет присвоены STREET_ID и BUILD_ID, и данные адреса будут выделены зеленым цветом. Адреса, для которых найдено более одного соответствия, подсвечиваются желтым цветом. Адреса, отсутствующие в базе данных, выделяются красным цветом.



Примечание:
1) Поиск в базе данных довольно длительная операция, поэтому наберитесь терпения.

6. Все строки выделенные желтым и красным можно отредактировать и повторить поиск повторным нажатием на кнопку «Поиск».

7. Для сохранения результатов работы текущей выбранной вкладки нажмите кнопку «Сохранить» и выберите место, имя и формат файла. Дождитесь завершения сохранения.

8. Для закрытия вкладки нажмите кнопку «Закрыть вкладку».
